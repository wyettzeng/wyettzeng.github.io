Siemens Healthineers

Android Software Engineer
Jan 2022 - Apr 2022, Ottawa, Canada

Siemens Healthineers is a medical device provider. During the role, I developed and maintained the in-house android program NXS for epoc Blood Analysis System’s host-4 devices using Android Studio and Java. This project employs an Object-Oriented Design that encapsulates functionalities into different modules. Some of my tasks include:

  • Implemented the eLabels function using Layout Inflator and ImageView API to display the host device’s hardware information and serial number for different countries and regions.
  • Fixed the Glucose conversion issue, which arose from incorrect asynchronous saving logic with Realm database and Reactive Java API.
  • Contacted Zebra Technologies’ technical support to expand our product’s compatibility with ZQ110 printer for several different languages.
  • Implemented NXS Helper application’s animation using AlphaAnimation API.
